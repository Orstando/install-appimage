Example usage (installs [Prism Launcher](https://prismlauncher.org/)): `sudo ./install-appimage -n "Prism Launcher" -d "Custom Minecraft Launcher to easily manage multiple Minecraft installations at once" -c "prism-launcher" -i PrismLauncher.ico ./PrismLauncher-Linux-x86_64.AppImage
`

See `./install-appimage --help` for details.

(also works for things that arent appimages, for example, it can install itself)
